Control posts per page for all your custom post types.

== Description ==
Lists all custom post types found on your site and gives you ability to change number of posts per page of each of them.

== Installation ==

1. Upload `posts-per-page` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
